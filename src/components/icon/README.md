# Icon

> 统一采用 svg 图标，图标源于 [ionicons](https://github.com/ionic-team/ionicons)

| 参数      | 说明                                       | 类型   | 可选值 | 默认值 |
| --------- | ------------------------------------------ | ------ | ------ | ------ |
| name      | icon 名字                                  | String | -      | -      |
| className | 自定义类名                                 | String | success、warn、error、自定义     | -      |
| color     | 单色图标颜色(svg 图标的 path 不设 fill 属性才生效) | String | -      | -      |
| size      | 自定义大小                                 | Number | -      | -      |
